在 NFS 环境下使能网络

Posted on 2018-11-05 Edited on 2020-07-11 In Linux Views:
Symbols count in article: 275 Reading time ≈ 1 mins.

挂载网络文件系统的同时可以使用 eth0 访问网络

当使用 nfs 时访问外部网络时会出现 Network is unreachable

需要添加路由表

route add -net 0.0.0.0 netmask 0.0.0.0 gw 192.168.110.1

或添加默认路由表

route add default gw 192.168.110.1 dev eth0

之后，可以访问 ip 地址，但是仍然不能访问网站，需要添加 nameserver

echo 'nameserver 192.168.110.1' > /etc/resolv.conf

经过上述两步修改之后，可以正常访问外部网络
