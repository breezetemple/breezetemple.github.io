Ubuntu 安装 Pandoc 和 TeX Live

Posted on 2020-09-27 Edited on 2021-02-06 In Tools , Markdown Views:
Symbols count in article: 1.3k Reading time ≈ 1 mins.

ubuntu18.04 下 pandoc + LaTeX 将 markdown 转为 pdf

Pandoc

$ sudo dpkg -r pandoc
$ sudo apt purge pandoc
$ wget https://github.com/jgm/pandoc/releases/download/2.10.1/pandoc-2.10.1-1-amd64.deb
$ sudo dpkg -i pandoc-2.10.1-1-amd64.deb
$ sudo apt install xzdec

TeX Live

TeX Live 是 LaTeX 的发行版，tlmgr 用于管理安装后的程序，包管理器

卸载

$ sudo apt purge texlive*
$ sudo rm -rf /usr/local/texlive/* and rm -rf ~/.texlive*
$ sudo rm -rf /usr/local/share/texmf
$ sudo rm -rf /var/lib/texmf
$ sudo rm -rf /etc/texmf
$ sudo apt remove tex-common --purge
$ rm -rf ~/.texlive
$ find -L /usr/local/bin/ -lname /usr/local/texlive/*/bin/* | sudo xargs rm

安装

apt

$ sudo apt install texlive-full

sh

  1. 安装
    $ wget https://mirrors.nju.edu.cn/CTAN/systems/texlive/tlnet/install-tl.zip
$ sudo perl install-tl
  2. 配置环境 ~/.zshrc
    export PATH=$PATH:/usr/local/texlive/2020/bin/x86_64-linux
export MANPATH=/usr/local/texlive/2020/texmf-dist/doc/man
export INFOPATH=/usr/local/texlive/2020/texmf-dist/doc/info
  3. -设置字体- 导致乱码
    $ sudo cp /usr/local/texlive/2020/texmf-var/fonts/conf/texlive-fontconfig.conf /etc/fonts/conf.d/09-texlive.conf
$ sudo fc-cache -fv

$ sudo rm sudo fc-cache -fv
$ sudo fc-cache -fv

4. 更新为国内源，可选
```shell
$ sudo visudo
# add /usr/local/texlive/2020/bin/x86_64-linux to secure_path
$ sudo tlmgr option repository https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/tlnet
  1. 更新
    $ sudo tlmgr update --list
$ sudo tlmgr update --self --all
